# Override proton prefix
```
TARGET
"/mnt/6TB/SteamLibrary/steamapps/compatdata/3231501569/pfx/drive_c/Program Files (x86)/Battle.net/Battle.net Launcher.exe"

START IN
/mnt/6TB/SteamLibrary/steamapps/compatdata/3231501569/pfx/drive_c/Program Files (x86)/Battle.net

LAUNCH OPTIONS
WINEPREFIX=/mnt/6TB/SteamLibrary/steamapps/compatdata/3231501569/pfx
```
